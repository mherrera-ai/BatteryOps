# Assets

These SVGs are the source visuals for the BatteryOps README and related public docs. They are the only committed exports; if you need raster assets, export them locally from the current SVGs instead of checking in stale PNG copies.

Current assets:

- `batteryops-hero.svg`: header graphic for the repository landing page
- `batteryops-pipeline.svg`: end-to-end workflow diagram used in the architecture section
- `batteryops-proof-card.svg`: compact provenance card for quick repository review

See the main [README](../../README.md) for where each asset appears in the public narrative.

If you change any of these source visuals, update the README and screenshot copy in the same change so the public docs stay aligned.
