# Security Policy

## Reporting Vulnerabilities

If you find a security issue, please report it privately through GitHub's security advisory flow for this repository. If that is not available, open a private direct message or contact the maintainer through the GitHub profile linked from the repository.

Please include:

- a short description of the issue
- the affected file, feature, or artifact
- clear reproduction steps
- any suggested mitigation, if you have one

## Scope

BatteryOps is a local-first Streamlit demo for public NASA battery data. This repository does not operate a hosted service, accept user accounts, or depend on a production backend in its default path.

Do not treat this project as production EV safety software. It is an engineering demo with saved artifacts, local workflows, and intentionally limited claims.
